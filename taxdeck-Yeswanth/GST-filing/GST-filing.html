Frequently asked questions

What if I am in a composite scheme?

Business falling under composite scheme can buy monthly package on the basis of number of entries required. For eg if you have sale entries of 50 per month. Your total will be 150 entries for the quarter. So you can buy the standard package which gives 100-200 entries

Who are required to file Annual return?

All taxpayers filing return in GSTR-1 to 3, which does not includes the casual taxpayers and taxpayers under composition scheme are required to file an annual return.

Is there any penalty for not filing the GST return?

All the GST number holders who does not file any GST Return will be tracked by the GST Department and a list of GST return defaulters will be provided to the respective GST authorities. This would incudes the imposition of the automatic late fee which is require to be paid by the GST holders.
